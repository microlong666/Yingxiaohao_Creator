# 沙雕营销号文案生成器

> 基于 [BullshitGenerator](https://github.com/menzi11/BullshitGenerator) 定制

### 特色

* 支持自定义字数，建议字数在 `400` 字以内

* [原生](https://github.com/menzi11/BullshitGenerator) 支持自动分段

* 对原版文字语法进行了优化，使语句更通畅（依然狗屁不通）
